インストール方法

投げる、即、編集。

インストーラーを開くだけで面倒な初期設定は一切不要です。※3
ただし、Python環境などの前提ソフトを既に導入しており、環境を汚さずに軽量に動かしたい開発者・プロの方は、GitHubからZip形式での導入を推奨します。

① 必須環境の確認

以下の環境・ライブラリが必要です（合計で約2GBの空き容量を要する場合があります）。

  • Python 3.10 〜 3.11 推奨
  • C++ & Python コンパイラ環境（VSCode / Xcode等）
  • 各必須ライブラリ (NumPy, sounddevice, soundfile, pyopenjtalk, mido, chardet, pytest, PySide6)

② 展開と実行

リポジトリからZipをダウンロードして解凍し、該当ディレクトリをエディタで開いて main.py を実行してください。Zip形式であっても面倒なセットアップスクリプトの実行は不要です。

※3 標準の .exe または .app 形式のインストーラーを利用する場合の注記です。
